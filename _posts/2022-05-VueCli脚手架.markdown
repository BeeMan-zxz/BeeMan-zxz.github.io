# VueCli

## 安装 VueCli

前提：需要安装 node 环境

全局安装 `npm install -g @vue/cli`

卸载 `npm uninstall vue-cli -g`

查看版本 `vue --version`

## 使用

### cmd命令创建

创建一个项目 `vue create 项目名称`



### 图形化界面创建

执行 `vue ui`
